API CSS

Le framework de composants intégre une API complète permettant de personnaliser le CSS de tous les composants. Chaque fois que vous rencontrez une propriété css sur un composant, la valeur de cette propriété doit être une liste de classes CSS séparées par des espaces.

Exemple :
sortlist.items[0]["css"] = "a b c" ajoutera les classes CSS a, b et c à l'élément d'index 0 de la liste triable (Sort List).

Le framework fournit plus de 80 classes CSS que vous pouvez utiliser pour personnaliser les composants. Les classes sont répertoriées dans la section ci-dessous.

Apparences

Cliquez sur une apparence pour copier la liste des classes css qui lui sont appliquées.

{{ css }}

Animations

Cliquez sur une animation pour l'appliquer aux boîtes de la section Apparences.

{{ animation }}

Options d'animation

Vous pouvez appliquer les options suivantes aux animations :

  • fast
  • faster
  • slow
  • slower
  • delay-1s
  • delay-2s
  • delay-3s
  • delay-4s
  • delay-5s
  • infinite
