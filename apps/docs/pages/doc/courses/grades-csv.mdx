# Téléchargement des notes

![Téléchargement des notes](../../../public/screens/courses/grades-csv.png)

Vous pouvez télécharger les notes des étudiants au format CSV.

## Télécharger les notes d'un cours

Pour télécharger les notes des étudiants, cliquez sur le bouton `Télécharger les notes au format CSV` en haut de la page du cours. *(Bouton **1** sur l'image ci-dessus)*

Une liste de cases à cocher apparaîtra pour vous permettre de sélectionner les groupes d'étudiants dont vous souhaitez télécharger les notes.

Cliquez ensuite sur le bouton `Télécharger` pour télécharger le fichier CSV.

## Télécharger les notes d'une activité

Pour télécharger les notes des étudiants pour une activité spécifique, cliquez sur le bouton `Télécharger les notes au format CSV` en bas de la carte d'une activité. *(Bouton **2** sur l'image ci-dessus)*

Une même liste de cases à cocher apparaîtra pour vous permettre de sélectionner les groupes d'étudiants dont vous souhaitez télécharger les notes.
