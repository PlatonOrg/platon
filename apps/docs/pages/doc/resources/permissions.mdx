# Gestion des droits

## Qui peut voir une ressource ?

Tous les enseignants et administrateurs peuvent voir les ressources sans restriction sauf si la ressource est dans un cercle personnel ou est un cercle personnel.

<Callout type="info">
  Il est possible de rendre visible les cercles personnels à des utilisateurs spécifiques. Pour cela, il faut ajouter
  les utilisateurs dans la liste des collaborateurs du cercle.
</Callout>

## Qui peut créer un cercle ?

Seuls les administrateurs peuvent créer des cercles.

## Qui peut créer/éditer des activités et exercices ?

Seuls les administrateurs et les utilisateurs définis comme "[collaborateurs](/doc/resources/members)" peuvent créer des activités et exercices.

## Qui peut modifier les informations d'une ressource ?

Seuls les administrateurs et les utilisateurs définis comme "[collaborateurs](/doc/resources/members)" peuvent modifier les informations d'une ressource.
