Annexes : Glossaire des Termes Utilisés

Voici un glossaire des termes employés tout au long de la documentation pour faciliter la compréhension et l'usage de la plateforme PLaTon :

- **Activité** : Ensemble structuré d'exercices que les apprenants doivent réaliser. Une activité peut être composée d'un ou plusieurs exercices.
- **API** : Interface de programmation d'applications (Application Programming Interface) qui permet la communication entre différents logiciels.
- **Builder** : Script ou partie de code dans PLaTon dédié à la construction d'un exercice, préparant les données avant que l'étudiant ne commence l'exercice.
- **Composant** : Élément d'interface utilisateur ou bloc fonctionnel utilisé dans la création d'exercices sur PLaTon.
- **Configurable** : Attribut d'un exercice qui peut être modifié ou personnalisé à l'aide de paramètres spécifiques.
- **Exercice** : Tâche ou problème posé aux apprenants pour évaluation ou pratique.
- **Grader** : Script ou partie de code qui évalue les réponses des apprenants et attribue des scores.
- **IDE** : Environnement de développement intégré (Integrated Development Environment) où les utilisateurs peuvent écrire et tester leur code.
- Navigation Dynamique : Capacité d'une activité à guider l'apprenant à travers les exercices de manière adaptative, basée sur son progrès ou ses réponses.
- **Player** : Composant au sein de PLaTon qui permet aux apprenants de visualiser et interagir avec les exercices et les activités.
- **Sandbox** : Environnement virtuel sécurisé où le code des exercices est exécuté isolément pour éviter tout effet indésirable sur le reste du système.
- **Session** : Période de temps pendant laquelle un apprenant est activement engagé dans un exercice ou une activité.
- **Template Nunjucks** : Langage de templating utilisé dans PLaTon pour insérer des variables et des logiques de programmation dans le contenu textuel d'un exercice.
- **Variable** : Élément de programmation qui stocke une valeur pouvant être modifiée et utilisée dans différents contextes au sein d'un exercice.
- **WYSIWYG** : Acronyme pour "What You See Is What You Get", désignant un éditeur qui permet de composer le contenu dans une forme qui ressemble à sa présentation finale.

Ce glossaire vise à fournir une référence rapide pour les termes techniques et spécifiques à PLaTon rencontrés par les utilisateurs lors de la création et de la gestion d'exercices et d'activités éducatives.
