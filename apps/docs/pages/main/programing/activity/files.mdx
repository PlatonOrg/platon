# Structure d'une activité

En plus d'éventuels fichiers de ressources (images, sons, etc.), une activité est composé des fichiers suivants :

- **main.pla** (PLaTon activity)
  - Fichier principal contenant le code de l'activité. Ce fichier ne peut pas être supprimé.
- **readme.md**
  - Document explicatif de l'activité, à usage des enseignants.

Lorsque vous créer une activité, ces fichiers sont automatiquement générés avec une activité vide. Vous pouvez ensuite les modifier à votre guise en suivant la suite de la documentation de ce module.
