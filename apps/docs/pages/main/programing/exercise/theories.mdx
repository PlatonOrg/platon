# Intégration de ressources théoriques

La clé `theories` dans PLaTon permet d'enrichir vos exercices avec des liens vers des ressources théoriques externes ou internes.
Cette fonctionnalité est essentielle pour fournir aux apprenants un contexte supplémentaire, des explications approfondies ou des références.
Voici comment vous pouvez intégrer efficacement des ressources théoriques dans vos exercices.

## Structure de la Clé theories

- La clé `theories` est un tableau d'objets, chaque objet représentant une ressource théorique.
- Chaque objet contient deux propriétés :
  - **title** : Un titre descriptif de la ressource.
  - **url** : L'URL de la ressource. Cela peut être un lien externe ou un lien interne vers un fichier hébergé sur la plateforme PLaTon.

## Exemple d'Utilisation

```js
theories = [
  { title: "Introduction à la programmation", url: "https://example.com/intro-programming" },
  { title: "Documentation sur les boucles", url: @copyurl /chemin/vers/la/doc/boucles.pdf }
]
```

Dans cet exemple, deux ressources théoriques sont ajoutées :

- La première est un lien externe vers un site web fournissant une introduction à la programmation.
- La seconde est un lien interne vers une ressource hébergée directement sur PLaTon, comme un fichier Markdown ou PDF.

# Mise en Œuvre dans l'Exercice

- Les liens fournis dans la clé `theories` seront accessibles aux apprenants, via le bouton `Théories` en bas dans l'interface de l'exercice.
- Lorsqu'un apprenant clique sur un lien, il est dirigé vers la ressource correspondante pour des informations supplémentaires dans un nouvel onglet si c'est un lien externe.
- Si le lien est interne, la ressource s'ouvre dans une boite de dialogue au dessus de l'exercice si c'est un fichier Markdown, PDF, image ou vidéo. Sinon le fichier se télécharge directement.

## Bonnes Pratiques pour l'Utilisation des Ressources Théoriques

- **Pertinence** : Assurez-vous que chaque ressource théorique est directement liée au contenu de l'exercice.
- **Accessibilité** : Vérifiez que les liens externes sont accessibles et que les ressources internes sont bien configurées.
- **Diversité des Sources** : Incluez une variété de ressources, comme des articles, des tutoriels vidéo, et des documents officiels, pour s'adapter aux différents styles d'apprentissage.
- **Mises à Jour** : Maintenez les liens à jour pour s'assurer que les apprenants accèdent à des informations actuelles et précises.
- **Titres Descriptifs** : Utilisez des titres clairs et informatifs pour chaque lien afin que les apprenants sachent à quoi s'attendre avant de cliquer.

L'intégration de ressources théoriques enrichit l'expérience d'apprentissage, en donnant aux apprenants les outils pour comprendre le contexte et les concepts clés de manière plus approfondie. Cette fonctionnalité est particulièrement utile pour créer des exercices plus interactifs et informatifs sur PLaTon.
