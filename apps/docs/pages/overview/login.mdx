# Connexion à la plateforme

Conçue pour être simple et sécurisée, PLaTon offre plusieurs méthodes de connexion adaptées aux besoins variés des utilisateurs.

Lorsque vous accédez à PLaTon, vous êtes accueilli par une page de connexion, invitant à entrer vos identifiants. Cette interface est conçue pour être intuitive, garantissant une expérience utilisateur fluide même pour ceux qui se connectent pour la première fois.


## Modes de Connexion

- **Pour les utilisateurs avec un compte PLaTon** : Vous devez entrer votre adresse e-mail et votre mot de passe associé.
Ce mode de connexion est destiné avant tout aux administrateurs de la plateforme. Pour les autres utilisateurs, il est recommandé de vous connecter avec votre LMS (voir ci-dessous).
![Page de connexion](../../public/screens/login/login.png)

- **Pour les utilisateurs avec un compte LMS** :  Si vous utilisez PLaTon dans le cadre d'un LMS comme Moodle ou Canvas, la connexion se fait sans effort grâce à l'intégration LTI.
En vous connectant à votre LMS habituel, vous trouverez un lien ou un bouton spécifique pour accéder directement à PLaTon.
Cette méthode simplifie l'accès en éliminant le besoin de se souvenir de multiples identifiants.

## Procédure de Connexion

- Pour les Comptes PLaTon :
  - Visitez la page de connexion de PLaTon.
  - Entrez vos identifiants de compte PLaTon (e-mail et mot de passe).
  - Cliquez sur 'Connexion' pour accéder à la plateforme.

- Pour les Comptes LMS via LTI :
  - Connectez-vous à votre LMS (Moodle, Canvas, etc.).
  - Recherchez l'intégration PLaTon ou le lien LTI dédié.
  - Cliquez sur le lien pour être automatiquement connecté à PLaTon.
